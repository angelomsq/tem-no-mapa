# SPEC - Tem No Mapa

## Project Overview
- **Name**: Tem No Mapa
- **Description**: Mapa global interativo mostrando onde vivem os brasileiros pelo mundo
- **Stack**: Vite + React + TypeScript + TailwindCSS + Leaflet + Supabase

## Core Features
1. Cadastro de usuários com localização
2. Mapa interativo com pontos de calor
3. Gamificação por estado
4. Sistema de compartilhamento viral
5. Rankings e conquistas

## Design
- Cor principal: Verde/amarelo (Brasil)
- Tipografia: Inter
- Mapa: OpenStreetMap via Leaflet
