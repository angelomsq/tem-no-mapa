A clean registration page where new users can create an account.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Theme: Light mode, clean and minimalist
- Background: White (#FFFFFF)
- Primary Accent: Brazilian Green (#009C3B) for submit button
- Secondary Accent: Brazilian Yellow (#FFDF00) for highlights
- Text Primary: Near Black (#1A1A1A) for headlines
- Text Secondary: Medium Gray (#6B7280) for labels
- Font: Plus Jakarta Sans for headlines, Be Vietnam Pro for body
- Roundness: 8px for buttons and inputs

**Page Structure:**
1. **Header:** Logo "Tem no Mapa" centered
2. **Register Card:** Centered form with name, email, password, confirm password inputs
3. **Terms Checkbox:** "I agree to the terms" checkbox
4. **Submit Button:** Primary green "Create Account" button
5. **Social Register:** "Or continue with Google" option
6. **Footer:** "Already have an account? Login" link
